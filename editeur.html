Dossier du dépôt

Sélectionne le dossier racine de ton dépôt local.

① Créer / Actualiser l'arborescence

Crée les dossiers manquants selon le schema. En mode actualisation, les dossiers absents du schema sont déplacés en _poubelle/.

② Publier sur GitHub Pages

Pousse tout le contenu du dossier local vers ton dépôt GitHub. Les fichiers > 50 Mo sont ignorés. La publication se fait par lots pour respecter les limites de l'API.
